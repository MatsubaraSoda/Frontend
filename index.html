关于

这个项目集合源自 GitHub 上的 app-ideas 仓库，目前包含 30+ 个初级的前端练习项目。完成这些项目的过程，帮助我熟悉了前端开发的基础流程。

大多数项目都是从零开始实现，没有使用框架，通过实践来理解原生前端三剑客 HTML CSS JS 的用法。

通过这些项目，我学到了：

  • 前端三剑客的基础用法
  • 把需求文档变成能跑的代码
  • 如何应对开发过程中的倦怠
  • 和 AI 有效协作的方式

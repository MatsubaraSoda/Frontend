# 笔记应用

**难度等级：** 1-初学者

创建并存储您的笔记以供日后使用！

## 用户故事

-   [ ] 用户可以创建笔记
-   [ ] 用户可以编辑笔记
-   [ ] 用户可以删除笔记
-   [ ] 当关闭浏览器窗口时，笔记将被保存；当用户返回时，数据将被检索

## 奖励功能

-   [ ] 用户可以创建和编辑 Markdown 格式的笔记。保存时会将 Markdown 转换为 HTML
-   [ ] 用户可以查看创建笔记的日期

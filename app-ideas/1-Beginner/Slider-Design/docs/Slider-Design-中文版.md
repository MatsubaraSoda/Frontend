# 滑块设计

**难度等级：** 1-初学者

使用滑块/轮播图展示多张图片。

## 用户故事

-   [ ] 用户可以每 `x` 秒看到滑块自动切换显示多张图片
-   [ ] 用户可以点击"上一张"和"下一张"按钮，滑块将显示对应的图片

## 额外功能

-   [ ] 为滑块添加动画效果
-   [ ] 在滑块上添加文字覆盖
-   [ ] 创建 3D 效果

## 有用的链接和资源

-   [Unsplash](https://unsplash.com/) 免费图片资源

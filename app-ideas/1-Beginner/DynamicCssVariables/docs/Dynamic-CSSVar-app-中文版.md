# 动态 CSS 变量

**难度等级:** 1-初学者

## 项目概述
创建一个登录表单，通过动态更改 CSS 变量来改变文本框的背景颜色，实现不同状态下的视觉反馈。

## 核心功能
- 用户 ID 和密码输入框
- 登录验证（用户名：testuser，密码：mypassword）
- 动态背景颜色变化：
  - 默认：白色
  - 包含空格：浅黄色
  - 验证失败：浅红色
- 取消按钮重置功能

## 技术要点
- CSS 变量（自定义属性）
- JavaScript 动态修改 CSS 变量
- 表单验证和状态管理

## 额外功能
- 错误状态下的边框颜色变化
- 错误状态下的字体样式变化
